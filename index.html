Project on Console

First, to use Visual Studio Code, you'll need to download it. Once downloaded and everything is set up, go to your file explorer and create a new folder. Once created, go back to Visual Studio Code, and click on file on the top left corner. Click open folder and select the file you recently created. Then you will see a blank page. On the left side in the box that says explorer at the top, right click and click new file and create index.html, style.css, and script.js. This is where you can start typing your code and you are ready to make whatever you want. To run your code, right click and click open with live server and it should work. Now you know how to use Visual Studio Code.

A console screen is something on your computer that allows a user to input commands and receive visual output from a computer or computer system. To view console screen, right click and click inspect. it will appear on your screen and on the top, click console. This is were you can view your commands.
